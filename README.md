# tools-cli
Development Tool

## help
USAGE:
    tools-cli [OPTIONS] [SUBCOMMAND]

OPTIONS:
    -h, --help             Print help information
    -o, --output <PATH>    Result output path
    -V, --version          Print version information

SUBCOMMANDS:
    help    Print this message or the help of the given subcommand(s)
    time    Time tools
